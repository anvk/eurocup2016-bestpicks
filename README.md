# EuroCup 2016 Best Picks

> My website/tool to calculate the most optimal pick for the EuroCup 2016 pool.

## How to Run Locally for Development

```
$ npm run dev
```

## How to Build Development Version

```
$ npm run build:dev
```

## How to Build Production Version

```
$ npm run build:prod
```
